1. Instalar o Pyhton 3.9

2. Instala Poetry comando abaixo
curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python -

3. Clonar o repositorio
git clone https://github.com/wborbajr/nfeimport.git

4. Entrar no diretorio nfeimport
cd nfeimport

5. Executar os comandos abaixo
poetry shell
poetry update
exit


Para executar manualmente
cd nfeimport
./NFeRun-Manual.sh
